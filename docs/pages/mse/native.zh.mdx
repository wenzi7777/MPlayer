## 播放原生视频


使用示例
```html
<script src="https://cdn.jsdelivr.net/npm/@wenzi7777/mplayer@latest/dist/mplayer.bundle.min.js"></script>
<div id={'mplayer'}></div>
<script>
    let options = {
        videos: [
            {
                title: 'video title',
                src: 'video src.mp4'
            }
        ]
    }
    const mplayer = new MPlayer(document.querySelector('#mplayer'), options)
</script>
```

原生视频**不支持**多分辨率
