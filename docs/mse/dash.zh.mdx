## 使用DASH插件

**若您需要播放mpd流视频，您需要在引入MPlayer前先引入dash.js库**

**[dash.js](https://github.com/Dash-Industry-Forum/dash.js)**

使用示例
```html
<script src="https://cdn.jsdelivr.net/npm/dashjs@4.5.1/dist/dash.all.debug.min.js"></script>
<script src="https://cdn.jsdelivr.net/npm/@wenzi7777/mplayer@latest/dist/mplayer.bundle.min.js"></script>
<div id={'mplayer'}></div>
<script>
    let options = {
        videos: [
            {
                title: 'video title',
                src: 'video src.mpd'
            }
        ]
    }
    const mplayer = new MPlayer(document.querySelector('#mplayer'), options)
</script>
```

**若您的mpd的manifests文件支持多分辨率，那么MPlayer会自动读取分辨率，提供给用户选择。**
