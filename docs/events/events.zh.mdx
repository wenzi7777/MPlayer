## 事件监听

**MPlayer提供了内建处理事件的API，您可以监听事件，然后加上事件处理函数。**

**api**
```javascript
const mplayer = new MPlayer(element, options)
mplayer.apis.on('eventName', () => {
    console.log('--')
})
```
